### Paper 확장 라이브러리

플러그인 제작에 자주 사용하는 기능을 확장합니다.

---

* #### Features
    * Config 파일 작성
    * ItemStack 제작 접근성 개선

* #### Supported minecraft versions
    * 1.17
    * 1.17.1
    * 1.18
    * 1.18.1
    * 1.18.2
    * 1.19
    * 1.19.1
    * 1.19.2
    * 1.19.3
    * 1.19.4
    * 1.20
    * 1.20.1
    * 1.20.2
    * 1.20.3
    * 1.20.4
    * 1.20.5
    * 1.20.6

---
